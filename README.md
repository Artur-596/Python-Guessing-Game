# Python-Guessing-Game
This is a console-based game written in Python, designed to practice core programming concepts including game loop logic, user input handling, and local file I/O.

Key FeaturesGame Logic: 
    Implements a randomized number generation system.
    Persistent Storage: Uses myfile.txt to save and retrieve the user's score, allowing progress to persist between sessions.
    Error Handling: Implements basic try...except blocks to handle file reading scenarios, ensuring the program does not crash if the score file is missing.

How to Run
    Ensure you have Python installed.
    Clone this repository to your local machine.
    Run the script using the command: python "Guessing Game2.0.py".

Future Improvements
